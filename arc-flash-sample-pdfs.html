Arc Flash Sample Studies

Download list for the extraction accuracy pass. Two things to feed SC: a text-extractable study report (tests the text path) and a scanned / image one-line (tests the vision path).

How SC reads each: a study PDF with a real text layer goes through the text path (pdfjs → AI). A scanned or vector-CAD one-line with no text layer now auto-rasterizes to an image and goes through the vision path — so you can upload either kind directly, no manual conversion. To force the vision path for a clean test, save a one-line page out of any PDF below as a PNG/JPG and upload that.

Study reports — for the TEXT path

Brady — Arc Flash Risk Assessment Sample Report BEST ALL-IN-ONE

~21 pp, SKM Power*Tools, 12.47 kV → 480Y/277 V. Has the one-line, the per-bus incident-energy table, and the IEEE 1584 inputs. This is the one SC already parsed — the primary text-path test.

e-Hazard — Arc Flash Study Report Example VERIFIED

IEEE 1584-2018 / NFPA 70E-2018 study, results from Level 4 down to <1.2 cal/cm². A second, differently-formatted report — good for testing the parser against a different layout.
