ServiceCycle Arc Flash Data Exchange — Format Source Index

Arc Flash Software Format Sources

A working index of links where we can download or view the real import/export data formats of the major arc-flash analysis and labeling tools. The goal: pull the actual CSV / Excel templates, documented column lists, and sample report PDFs from each vendor, then map their schemas against the open AFX (Arc Flash Data Exchange) standard we are designing on the IEEE 1584 / NFPA 70E data model. Grab the files tagged HIGH first — those are actual templates, column lists, or real export samples we can reverse-engineer into column mappings. Items tagged MED are docs that describe the workflow/columns without a file; LOW is marketing/context only. Feed the downloaded files back in and we will extract a column-by-column crosswalk per tool.

Legend. File-type chips: XLSX CSV ZIP PDF HTML/DOC. Value: HIGH actual template/column list/sample file · MED documented workflow/columns · LOW marketing/context.
