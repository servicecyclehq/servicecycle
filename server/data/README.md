# server/data

Generated artifacts that ship inside the lapseiq-server docker image.

## LapseIQ_AI_GUIDE.txt

A copy of `docs/LapseIQ_AI_GUIDE.md`, synced via `npm run guide:sync`.
Do NOT edit this file directly — edits will be overwritten by the next
sync. Author the guide in `docs/LapseIQ_AI_GUIDE.md` and run the sync
script before committing.

The `.txt` extension is intentional: `server/.dockerignore` excludes
`*.md` to keep stray notes out of the runtime image, and we don't
want the AI Guide caught in that net.
